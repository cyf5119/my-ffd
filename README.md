# FFD绝欧米茄P3绘制  
**零基础土制触发**  
**有bug请直接反馈**
## 目前功能
+ P2.5 步进月环绘制，FFD自带绘制手  
  开启选项后可绘制方位连线提示
+ P3 HW 大圈分摊绘制
+ P3 HW 指路去哪
+ P3 带毒提醒
+ P3 炸毒范围提示
+ P3 小电视点位绘制  
  开启选项后绘制连线提示与**箭头提示面向**  
  日基无脑固定式点位  
## 使用方法
+ 丢入FFD的plugins文件夹  
+ 若使用优先级相关内容，  
  需在进本前在 欧米茄绝境验证战/绝欧P3 中勾起选项  
  并在Raid Helper/party role中正确设置各队员位置
